The scan head of the LiSPM is relatively simple. All it has to do is keep the following parts in place:

- Piezo disk, at a 90º to the linear slider
- Capacitive sensor, with design space for the adjusting screws
- TIA board, as close to the tip as possible

The piezo support holds the piezo disk, which holds the standoff, which holds the tip socket, which holds the tip. The standoff also supports the capacitive distance sensor inner metal disk, so it can move between the two sensor PCB’s.
